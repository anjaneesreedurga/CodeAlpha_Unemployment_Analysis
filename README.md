# Unemployment Analysis in India using Python

## Project Overview

This project focuses on analyzing unemployment trends in India using Python. The dataset was cleaned, processed, and visualized to identify patterns, regional differences, and the impact of COVID-19 on unemployment rates.

The project demonstrates the use of data analysis techniques, exploratory data analysis (EDA), and data visualization to derive meaningful insights from real-world economic data.

---

## Objectives

* Clean and preprocess unemployment data.
* Analyze unemployment trends across different states.
* Compare unemployment rates in rural and urban areas.
* Study the impact of COVID-19 on unemployment.
* Visualize key trends using charts and graphs.
* Generate actionable insights from the dataset.

---

## Technologies Used

* Python
* Pandas
* Matplotlib
* CSV Dataset

---

## Dataset Features

The dataset contains the following attributes:

* Region
* Date
* Frequency
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)
* Area (Urban/Rural)

---

## Project Workflow

### 1. Data Cleaning

* Removed unnecessary spaces from column names.
* Converted date values into datetime format.
* Handled missing values by removing incomplete records.

### 2. Exploratory Data Analysis (EDA)

* Examined dataset structure.
* Checked data types and statistical summaries.
* Identified missing values.

### 3. State-wise Analysis

* Calculated average unemployment rates for each state.
* Identified states with the highest and lowest unemployment rates.

### 4. Area-wise Analysis

* Compared unemployment rates between Urban and Rural regions.
* Visualized area-wise unemployment trends.

### 5. Time Series Analysis

* Analyzed monthly unemployment trends.
* Identified unemployment spikes during the COVID-19 lockdown period.

### 6. Correlation Analysis

* Studied relationships among:

  * Unemployment Rate
  * Employment
  * Labour Participation Rate

## Key Findings

### COVID-19 Impact

* Unemployment remained relatively stable before the pandemic.
* A sharp increase was observed during April and May 2020.
* The highest unemployment rate was recorded in May 2020.

### Highest Unemployment States

* Tripura
* Haryana
* Jharkhand
* Bihar
* Himachal Pradesh

### Lowest Unemployment States

* Meghalaya
* Odisha
* Assam
* Uttarakhand
* Gujarat

### Area-wise Observation

* Urban areas showed higher unemployment rates than rural areas.

### Correlation Insights

* Employment and unemployment have a negative relationship.
* Labour participation rate showed a weak correlation with unemployment.

## Results

The analysis successfully highlighted the effects of COVID-19 on unemployment trends and demonstrated how data analytics can be used to extract valuable economic insights.
